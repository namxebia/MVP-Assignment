# MVP-Assignment

## Step1: Clone project:
```
git clone https://github.com/namxebia/MVP-Assignment.git
```
## Step2: Go to project:
```
cd MVP-Assignment/
```

## Step3: Installation:
### For android: 
Make sure you run the android emulator already
```
npm run android
```
### For ios: 
```
cd ios/
pod install
cd ../
npm run ios
```
